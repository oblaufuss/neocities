USEFUL STUFF AROUND THE WEB...

SOFTWARE I USE

  • Audacity (A simple, free, open source audio editing/recording software)
  • Krita (A free, open source art program with great brushes, stabilizers, and scripting)
  • OBS Studio (Free, open source streaming that I use for Telephony Cacophony.)
  • Sublime Text (The text editor I use for this site. The free trial can be indefinitely extended.)
  • VLC Media Player (Free, open source video player and converter.)
  • Ubuntu (I'm no OS snob, but had to use 18.04 for class and have gotten comfortable.)

SITE TOOLS

  • Base 64-image (Convert images into base 64 to optimize in CSS)
  • Brandmark pallete generator (Try cool AI-generated alternate colorings on images)
  • Butterick's Practical Typography (Good advice on making websites/documents readable and professional
  • Bytenbit or Labnol (embed Google Photos pics directly into a website)
  • Comma Delimiter (Format a lot of data, fast and with lots of useful settings!)
  • Dr. Link Checker (Scan your neocities for broken links)
  • Material.io (Color, typographic, and design tools for websites & apps. Huge library of free icons.)
  • Shapecatcher (Find the Unicode symbol you want by drawing it.)
  • TinyPNG (Optimize .png or .jpg files by a lot.)
  • Wayback Machine (See webpages as they were all the way back to the late 90s.)
